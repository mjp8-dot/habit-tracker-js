# Habit Tracker

Habit Tracker is a JavaScript-powered productivity web application that helps users build consistency through daily habit management. Users can create habits, mark them as completed, track progress visually, and save their data using browser local storage.

## Features

* Add and remove habits
* Mark habits as completed
* Progress tracking bar
* Dark and light themes
* Local storage persistence
* Dynamic DOM manipulation
* Interactive user interface

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Local Storage API

## Learning Outcomes

This project was built to practice core JavaScript concepts including:

* DOM manipulation
* Event handling
* Dynamic element creation
* Local storage management
* State updates
* User interaction design
